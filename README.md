### pyrootplots

#### Documentation

https://pyrootplots.readthedocs.io/en/latest/

#### Development

Setting up the environment:

```shell
conda env create -f environment.yml
conda activate pyrootplots
```

Editing:

```shell
cd src
```

Building the docs:

```shell
cd docs
make html
```
